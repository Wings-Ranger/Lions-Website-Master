# Problems Reports Index

This directory contains issue reports for the Lions website source files. Reports are organised as shared cross-file reports and individual per-file reports.

## Cross-file reports

These cover issues that appear across multiple files:

| File | Scope |
|---|---|
| `html.md` | Issues common to all 10 HTML pages |
| `css.md` | Issues across CSS files |
| `js.md` | Issues in JavaScript architecture |

## How reports are structured

- Each per-file report lists file-specific issues and references the relevant cross-file reports for shared issues.
- Issues include a description, explanation of impact, and a suggested improvement.
- A quick maintenance checklist at the bottom of each report summarises actionable items.
