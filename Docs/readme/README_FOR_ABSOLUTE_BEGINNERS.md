# Harcourt Lions Website Guide - For Absolute Beginners

This guide is for people with no tech background.

If you can open a file and type words, you can use this guide.

## What this website is

The website is a set of pages.
Think of each page like one sheet in a folder.

Main pages:
- Home page
- About Us page
- Membership page
- Meetings page
- Contact Us page
- Gallery page
- Projects page
- Resources page
- Site Map page
- 404 (page not found) page

## Where the website files are

In the website folder, these are the important files:
- Home.html
- about-us.html
- membership.html
- meetings.html
- contact-us.html
- gallery.html
- projects.html
- resources.html
- 404.html
- sitemap.html
- site.js (adds the menu and footer to every page)
- style.css
- images folder (this stores pictures)

## Before you change anything (important)

1. Make a copy of the whole website folder.
2. Rename the copy to include today's date.
3. Only edit one thing at a time.

If something goes wrong, use your backup copy.

## How to open the website

1. Open the website folder.
2. Double-click Home.html.
3. Your browser will open the website.

## How to change text on a page

1. Right-click the page file you want to edit. Example: about-us.html.
2. Open with Notepad or VS Code.
3. Find the words you want to change.
4. Replace only the words.
5. Save the file.
6. Go back to the browser and refresh the page.

Tip:
- Keep symbols like < and > exactly as they are.
- Only change the words between them.

## How to change a picture

1. Open the images folder.
2. Add your new picture file.
3. If possible, use the same file name as the old picture.
4. Refresh the website in your browser.

If the image does not appear, the file name may not match.

## Safe editing rules

- Do not rename files unless you really need to.
- Do not move files into different folders.
- Do not delete files you are unsure about.
- Save after each small change.
- Refresh the browser after each save.

## What to do if a page looks broken

1. Close the file without more edits.
2. Re-open the file.
3. Undo the last change.
4. Save again.
5. Refresh browser.

If still broken, replace the file with the backup copy.

## Quick troubleshooting

Problem: Menu links stop working.
- A file name may have changed.
- Put the original file name back.

Problem: Page has no styling.
- style.css may be moved or renamed.
- Put style.css back in the main website folder.

Problem: Image is missing.
- Check the image file is in the images folder.
- Check spelling of the image file name.

## 5-minute update checklist

1. Open website from Home.html.
2. Edit one page file.
3. Save.
4. Refresh browser.
5. Check desktop view and mobile view.
6. If all good, keep your updated version.

You do not need coding knowledge to do basic content updates safely.
